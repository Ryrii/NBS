Clonez ce dépôt et installez les dépendances :

```bash
git clone https://github.com/Ryrii/NBS
cd NBS
npm install

Développement
Pour démarrer l'application en mode développement :
npm start

Compilation

WINDOWS :

npm run build -- --win

Les fichiers compilés se trouveront dans le dossier dist/win-unpacked.

MACOS :

npm run build -- --mac

Les fichiers compilés se trouveront dans le dossier dist/mac.

LINUX :


npm run build -- --linux

Les fichiers compilés se trouveront dans le dossier dist/linux-unpacked.
